1) FileReaderWithExceptions
Консольное приложение на C# для чтения содержимого текстового файла с корректной обработкой исключений.  
Разработано в рамках практического задания по МДК.01.04 «Системное программирование».

2) Требования
- .NET SDK 6.0 или выше
- Операционная система Windows / Linux / macOS (поддерживается .NET Core)

3)  Запуск

4) Через командную строку (dotnet CLI)
	1. Откройте терминал в папке проекта (где находится файл `.csproj`).
	2. Выполните команду:
   		cmd
		  cd путь к директории
   		dotnet run
